Introduction

RANS is library, implementation of Abstract Numeration System (ANS) on a regular language.

Quick Start

$ git clone "http://sinya8282@github.com/sinya8282/RANS.git"
$ cd RANS
$ make check        # can be skipped
$ make install      # install only header
$ make install-all  # or install both header & RANS cui (rans)

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Data Compression

Here I give a concrete example of engineering application about data compression using RANS. that is, http-url.

formal http-url syntax can be defined by the (POSIX extended) regular expression. such that: 

http://((([a-zA-Z0-9]|[a-zA-Z0-9][-a-zA-Z0-9]*[a-zA-Z0-9])\\.)*([a-zA-Z]|[a-zA-Z][-a-zA-Z0-9]*[a-zA-Z0-9])\\.?|[0-9]+\\.[0-9]+\\.[0-9]+\\.[0-9]+)(:[0-9]*)?(/([-_.!~*'()a-zA-Z0-9:@&=+$,]|%[0-9A-Fa-f][0-9A-Fa-f])*(;([-_.!~*'()a-zA-Z0-9:@&=+$,]|%[0-9A-Fa-f][0-9A-Fa-f])*)*(/([-_.!~*'()a-zA-Z0-9:@&=+$,]|%[0-9A-Fa-f][0-9A-Fa-f])*(;([-_.!~*'()a-zA-Z0-9:@&=+$,]|%[0-9A-Fa-f][0-9A-Fa-f])*)*)*(\\?([-_.!~*'()a-zA-Z0-9;/?:@&=+$,]|%[0-9A-Fa-f][0-9A-Fa-f])*)?)?

This is hairy, pathological expression. However, the minimal DFA that corresponds this has only 26 states. I generated one graph of this DFA as shown below.

val("") on S equals to 872522588361473837 (base 10) , and data size of this http-url and number is 17 and 8 (bytes). that is, I think ANS can give us "compact" representation of strings on the assumption that we know well-defined language (or scheme). like a http-url, there are many scheme based languages in real-world.

Is RANS works well on Big Data?

No, I don't think so.
because the larger the number (which is calculated in val()) becomes, the more operation and space will be needed in multiprecision integer multiplication.
